About This Souhail Sabri Game

The Drone Assault Arcade Game is an interactive browser game project created by Souhail Sabri. This page is part of the official Souhail Sabri website and showcases a playable JavaScript canvas game with arcade-style action.

Souhail Sabri built this web game to demonstrate front-end development, browser-based interactivity, animation, keyboard controls, mouse aiming, collision detection, enemy behavior, and real-time scoring. The project combines creative game design with technical JavaScript programming.

Game Features

  • Browser-based JavaScript canvas gameplay
  • Drone and helicopter-style enemy combat
  • Keyboard movement and mouse aiming controls
  • Score tracking and wave progression
  • Replayable arcade-style action

How to Play Drone Assault Arcade

Use the keyboard to move your character across the battlefield and use the mouse to aim. The goal is to survive waves of enemy aircraft, shoot down drones and helicopters, and earn the highest score possible.

This Souhail Sabri arcade game runs directly in the browser, so no download is required. The game uses HTML5 canvas and JavaScript to draw the player, enemies, projectiles, explosions, score display, and game-over screen.
